Overview

Android100 is a hilarious project based on Android operating system, designed to run on web browsers. It is a parody of the Android ecosystem, featuring a collection of built-in apps and a unique user interface that mimics the look and feel of Android devices. Taking an inspiration from Windows Doors and Windows 93, and bringing the all memes and jokes to mobile platform. So welcome to Android 100.

Project Structure

  • /program Android100 built-in apps were stored here.
  • /system Storing crucial files, such as launcher, icons, etc

Features

  • A working camera
  • Hilarious chatbot
  • Working Browser
  • Calculator
  • Mini Games such as SquareFight

Here are the minimum browsers required to run Android100:
